# Gym Trains 🏋️🔥

**Gym Trains** is a mobile-first hypertrophy tracking and workout optimization SPA. Built for lifters who want to log their training volume, track their progressive overload, and optimize their muscle gains.

## Features

- **Supplementation Intake** — Secure access to your personalized training database
- **Workout Configuration** — Choose your set count (5, 10, or full session)
- **Execution Arena** — Real-time tracking of your target weights and total training volume
- **Performance Analytics** — Post-workout breakdown of your hypertrophy gains and muscular rendimiento

## Tech Stack

- Tailwind CSS — Dark, neon-accented UI
- CryptoJS — AES-256 encrypted routine data
- Vanilla JavaScript — No dependencies, instant load

## Getting Started

1. Open `index.html` in any browser
2. Enter your pre-workout formula to load your routine
3. Select your desired set count
4. Start tracking your exercises
